# 水墨流韵 — 水拓丹青互动画布

> 一池宣纸做的水面，指尖一划，墨随水走。

基于 GPU 流体模拟的中国风水拓画（水影画）互动网页：墨滴落入水中，晕开、打旋、随波流动，如同真实的墨在水面化开。

单 HTML 文件，零依赖、无需构建、可完全离线，浏览器打开即玩。

## 功能

- **指尖落墨** — 按住拖动，墨随笔走；悬停轻晃只拨水不落墨，已有墨纹随波荡漾
- **五种中国传统色** — 玄墨、黛青、朱砂、竹青、藤黄；「轮转」模式下颜色随时间在五色间流转
- **自动演墨** — 闲置时自动滴墨、自起暗流，一幅会动的水墨屏保
- **制扇** — 把当前水面的墨纹拓到一把折扇上（数字版"漆扇"），可保存为高清 PNG
- **涤净** — 一池浑水徐徐洗净，从头再来

## 本地运行

```bash
# 直接用浏览器打开
open 水墨流韵/index.html

# 或起一个本地服务器
python3 -m http.server 8741 --directory 水墨流韵
```

## 技术

- 原生 WebGL 实现 Stable Fluids 流体模拟（平流 → 涡度增强 → 压力投影），零依赖
- 墨色以**吸光度**累积，显示时按 `宣纸色 × exp(-A)` 做减法混色，墨色叠加如真实颜料
- 速度场 256、墨色场 1280 分辨率，桌面与移动端均可流畅运行
- 支持触屏，`prefers-reduced-motion` 下自动关闭动效
